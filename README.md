# Weather Forecast Exercises

This repository contains my solutions for the JavaScript weather-related exercises.

## 📚 Resources

The exercises and data sets used in this project are based on the Gist by **Jordan Castelloe**:

- [Original Exercise Link](https://gist.github.com/jordan-castelloe/3eb462ad073457a7ab4190ffcd770409)

## 🛠️ Requirements & Logic

- All exercises are solved using **Pure JavaScript**.
- **No external libraries** were used for data calculation in Exercise 2.
- Calculations for **Average** and **Median** were implemented manually to demonstrate logical thinking.

## 🚀 How to Run

1. Clone the repository: `git clone https://github.com/noahtran96/weather-forecast.git`
2. Run any file using Node.js: `node filename.js` (e.g., `node weather_data.js`)
